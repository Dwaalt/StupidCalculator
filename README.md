Stupidly simple calculator what counting elementary arithmetic expressions.

When using it, please note the increasing error, it is better to use numbers smaller than 10^16.

Supports:

[+] sum

[-] minus and unary minus

[*] multiplication

[/] division

[**] ascending

[()] - parentheses
